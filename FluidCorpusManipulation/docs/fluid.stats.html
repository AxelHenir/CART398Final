fluid.stats

Rolling mean and standard deviation on control inputs

Description

Computes the rolling mean and sample standard deviation over a given window for multichannel control inputs.

Discussion

Each element of the list is dealt with in parallel. This is particularly suitable for multidimensional descriptors.

Read more about fluid.stats on the learn platform.

fluid.stats is part of the Fluid Decomposition Toolkit of the FluCoMa project. For more explanations, learning material, and discussions on its musicianly uses, visit flucoma.org.

Arguments

Input List Size [number]

Optional
(default: 32)

number of items in the input list

Attributes

autosize [number] (default: 1)

If the input list is a different size to inputsize then automatically resize the internal data. If the object is triggered from the high priority thread and a resize is needed, then the call will be deferred to the low priority thread and a warning will be posted to the console.

history [number] (default: 2)

The size of the history window to use.

Constraints

  • Minimum: 2

warnings [number] (default: 0)

Enable warnings to be issued whenever a parameter value is constrained (e.g. clipped)
