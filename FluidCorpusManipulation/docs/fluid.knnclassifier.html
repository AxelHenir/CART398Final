fluid.knnclassifier

Classification with K-Nearest Neighbours

Description

A nearest neighbour classifier using a fluid.kdtree .

Discussion

In order to make predictions, the KNNClassifier must first be fit with a fluid.dataset of data points and a target fluid.labelset with a label for each point in the DataSet (by means of a shared identifier).

To classify a point, numneighbours neighbours are determined for the incoming point and whichever class is most common among the neighbours is predicted as the class for the point. If an even number of numneighbours is requested and there is a tie, the label with the closer point will be predicted.

Read more about fluid.knnclassifier on the learn platform.

fluid.knnclassifier is part of the Fluid Decomposition Toolkit of the FluCoMa project. For more explanations, learning material, and discussions on its musicianly uses, visit flucoma.org.

Arguments

Name [number]

Optional
(default: )

Name for this instance, so that it can be reused or referred to by other data objects. If none is given, a name will be generated

Attributes

numneighbours [number] (default: 3)

The number of neighbours to consider

Constraints

  • Minimum: 1

warnings [number] (default: 0)

Enable warnings to be issued whenever a parameter value is constrained (e.g. clipped)

weight [int] (default: 1)

Whether the neighbours should be weighted by their distance so that closer points have more influence over determining the class. The default is 1 (true).

Messages

fit

Arguments: dataSet (Unknown Type (input dataset)), labelSet (Unknown Type (input labelset))

Fit the model to a source fluid.dataset and a target fluid.labelset. The labels in the fluid.labelset correspond to the data points in the fluid.dataset by means of a shared identifier.

Argument details:
  • dataSet:

    Source fluid.dataset

  • labelSet:

    A fluid.labelset of labels for the source dataset

predict

Arguments: dataSet (Unknown Type (input dataset)), labelSet (symbol)

Given a fitted model, predict labels for a fluid.dataset and write these to a fluid.labelset

Argument details:

    A fluid.dataset of data points to predict labels for

  • labelSet:

    A fluid.labelset to write the predicted labels into
