fluid.gain~

Realtime Buffered Gain Changer

Description

This class implements a sanity test for the FluCoMa Realtime Client Wrapper.

Discussion

Read more about fluid.gain~ on the learn platform.

fluid.gain~ is part of the Fluid Decomposition Toolkit of the FluCoMa project. For more explanations, learning material, and discussions on its musicianly uses, visit flucoma.org.

Arguments

Attributes

gain [number] (default: 1.0)

Audio or control rate change of the gain.

warnings [number] (default: 0)

Enable warnings to be issued whenever a parameter value is constrained (e.g. clipped)

Messages

reset

Calling reset will change parameter values back to those the object was created with, i.e. any attribute values set in the box will be retained
