fluid.labelset

A set of labels associated with identifiers.

Description

FluidLabelSet is a container associating identifiers with labels.

Discussion

Read more about fluid.labelset on the learn platform.

fluid.labelset is part of the Fluid Decomposition Toolkit of the FluCoMa project. For more explanations, learning material, and discussions on its musicianly uses, visit flucoma.org.

Arguments

Name of the LabelSet [number]

Optional
(default: )

Symbol with the FluidLabelSet name.

Attributes

warnings [number] (default: 0)

Enable warnings to be issued whenever a parameter value is constrained (e.g. clipped)

Messages

addlabel

Arguments: identifier (symbol), label (symbol)

Add a label to the FluidLabelSet.

Argument details:
  • identifier:

    The identifier for this label.

  • label:

    The label to add.

getlabel

Arguments: identifier (symbol)

Retrieve the label associated with an identifier. Will report an error if the identifier is not present in the set.

Argument details:
  • identifier:

    The identifier for the label to be retrieved.

deletelabel

Arguments: identifier (symbol)

Delete a label given a certain identifier.

Argument details:
  • identifier:

    The identifier to be deleted.

updatelabel

Arguments: identifier (symbol), label (symbol)

Update a label for a given identifier in the label set.

Argument details:
  • identifier:

    The identifier for this label.

  • label:

    The label to update.

setlabel

Arguments: identifier (symbol), label (symbol)

Set a label for a given identifier.

Argument details:
  • identifier:

    The identifier for this label.

  • label:

    The label to update.

merge

Arguments: sourceLabelSet (symbol), overwrite (int)

Merge sourceLabelSet in the current LabelSet. It will replace the label with the same identifier if overwrite is set to 1.

Argument details:
  • sourceLabelSet:

    The source LabelSet to be merged.

  • overwrite:

    A flag to allow overwrite labels with the same identifier.

print

Arguments:

Post an abbreviated content of the label set in the window by default, but you can supply a custom action instead.

    size

    Arguments:

    The number of data points (entries / observations) in this model or dataset / labeset

      cols

      Arguments:

      The number of columns (dimensions) in this model or dataset / labeset

        clear

        Arguments:

        Empty the label set and reset to a clean state.

          write

          Arguments path (symbol, optional)
          Save the internal state of the object to a JSON file on disk. With no argument, a file dialog will appear to choose the file.

          read

          Arguments path (symbol, optional)
          Replace the internal state of the object from a JSON file on disk. With no argument, a file dialog will appear to choose the file.

          getids

          Arguments: labelSet (symbol)

          Export the LabelSet identifier to a FluidLabelSet.

          Argument details:
          • labelSet:

            The FluidLabelSet to export to. Its content will be replaced.

          reset

          Calling reset will change parameter values back to those the object was created with, i.e. any attribute values set in the box will be retained
