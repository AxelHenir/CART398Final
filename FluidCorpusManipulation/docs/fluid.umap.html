fluid.umap

Dimensionality Reduction with Uniform Manifold Approximation and Projection

Description

Reduce the dimensions of a fluid.dataset using the Uniform Manifold Approximation and Projection (UMAP) algorithm.

Discussion

Performs dimensionality reduction of a fluid.dataset using Uniform Manifold Approximation and Projection (UMAP)

Please refer to https://umap-learn.readthedocs.io/ and https://learn.flucoma.org/reference/umap for more information on the algorithm.

Read more about fluid.umap on the learn platform.

fluid.umap is part of the Fluid Decomposition Toolkit of the FluCoMa project. For more explanations, learning material, and discussions on its musicianly uses, visit flucoma.org.

Arguments

Name [number]

Optional
(default: )

Name for this instance, so that it can be reused or referred to by other data objects. If none is given, a name will be generated

Attributes

iterations [number] (default: 200)

The number of iterations that the algorithm will go through to optimise the new representation

Constraints

  • Minimum: 1

learnrate [number] (default: 0.1)

The learning rate of the algorithm, aka how much of the error it uses to estimate the next iteration.

Constraints

  • Minimum: 0.0
  • Maximum: 1.0

mindist [number] (default: 0.1)

The minimum distance each point is allowed to be from the others in the low dimension space. Low values will make tighter clumps, and higher will spread the points more.

Constraints

  • Minimum: 0

numdimensions [number] (default: 2)

The number of dimensions to reduce to

Constraints

  • Minimum: 1

numneighbours [number] (default: 15)

The number of neighbours considered by the algorithm to balance local vs global structures to conserve. Low values will prioritise preservation of the local structure while high values will prioritise preservation of the global structure.

Constraints

  • Minimum: 1

warnings [number] (default: 0)

Enable warnings to be issued whenever a parameter value is constrained (e.g. clipped)
