fluid.dataset

A set of data associated with identifiers.

Description

FluidDataSet is a container associating data points with identifiers.

Discussion

Read more about fluid.dataset on the learn platform.

fluid.dataset is part of the Fluid Decomposition Toolkit of the FluCoMa project. For more explanations, learning material, and discussions on its musicianly uses, visit flucoma.org.

Arguments

Name of the DataSet [number]

Optional
(default: )

The name of the FluidDataSet. This is unique between all FluidDataSets.

Attributes

warnings [number] (default: 0)

Enable warnings to be issued whenever a parameter value is constrained (e.g. clipped)

Messages

addpoint

Arguments: identifier (symbol), buffer (symbol)

Add a new point to the FluidDataSet. The dimensionality of the FluidDataSet is governed by the size of the first point added. If the identifier already exists, or if the size of the data does not match the dimensionality of the FluidDataSet an error will be reported.

Argument details:
  • identifier:

    The identifier for the point.

  • buffer:

    A array containing the data for the point.

getpoint

Arguments: identifier (symbol), buffer (symbol)

Retrieve a point from the data set into a array. If the identifier does not exist an error will be reported.

Argument details:
  • identifier:

    The identifier for the point to be retrieved.

  • buffer:

    A array where the retrieved data will be stored.

setpoint

Arguments: identifier (symbol), buffer (symbol)

Set the point. If the identifier exists, this method behaves like updatePoint. If the identifier doesn't exist, it behaves like addPoint.

Argument details:
  • identifier:

    The identifier for this point.

  • buffer:

    A array containing the data for the point.

updatepoint

Arguments: identifier (symbol), buffer (symbol)

Update an existing identifier's data. If the identifier does not exist, or if the size of the data does not match the dimensionality of the FluidDataSet an error will be reported.

Argument details:
  • identifier:

    The identifier for this point.

  • buffer:

    A array containing the data for the point.

deletepoint

Arguments: identifier (symbol)

Remove a point from the data set. If the identifier doesn't exist an error will be reported.

Argument details:
  • identifier:

    The identifier to be deleted.

merge

Arguments: sourceDataSet (Unknown Type (input dataset)), overwrite (int)

Merge sourceDataSet in the current DataSet. It will replace the value of points with the same identifier if overwrite is set to 1. ​To add columns instead, see the 'transformJoin' method of FluidDataSetQuery.

Argument details:
  • sourceDataSet:

    The source DataSet to be merged.

  • overwrite:

    A flag to allow overwrite points with the same identifier.

          frombuffer

          Arguments: buffer (symbol), transpose (int), labelSet (Unknown Type (input labelset))

          Import to the dataset the content of a array, with optional transposition, and a map of frames/channels to the original IDs as a FluidLabelSet.

          Argument details:
          • buffer:

            The buffer to read from. The dataset will be resized.

          • transpose:

            If 0, each buffer frame becomes a dataset point, and each buffer channel becomes a dataset dimension. If 1, channels become points, and frames become dimensions.

          • labelSet:

            The FluidLabelSet from which to retrieve the point's IDs associated with their reference frame number (or channel number if transposed).

          tobuffer

          Arguments: buffer (symbol), transpose (int), labelSet (symbol)

          Dump the content of the dataset to a array, with optional transposition, and a map of frames/channels to the original IDs as a FluidLabelSet.

          Argument details:
          • buffer:

            The buffer to write to. It will be resized.

          • transpose:

            If 0, each dataset point becomes a buffer frame, and each dataset dimension becomes a buffer channel. If 1, points become channels, and dimensions become frames.

          • labelSet:

            The FluidLabelSet in which to dump the point's IDs associated with their reference frame number (or channel number if transposed).

          getids

          Arguments: labelSet (symbol)

          Export the dataset identifier to a FluidLabelSet.

          Argument details:
          • labelSet:

            The FluidLabelSet to export to. Its content will be replaced.

          knearest

          Arguments: buffer (symbol), k (int)

          Returns the identifiers of the k points nearest to the one passed. Note that this is a brute force distance measure, and comparatively inefficient for repeated queries against large datasets. For such cases, fluid.kdtree will be more efficient.

          Argument details:
          • buffer:

            A array containing a data point to match against. The number of frames in the buffer must match the dimensionality of the DataSet.

          • k:

            The number of nearest neighbours to return. The identifiers will be sorted, beginning with the nearest.

          reset

          Calling reset will change parameter values back to those the object was created with, i.e. any attribute values set in the box will be retained
