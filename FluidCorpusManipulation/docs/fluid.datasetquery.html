fluid.datasetquery

Query a FluidDataSet

Description

A selection of columns and a set of conditions that match rows of a FluidDataSet. Used to filter and search in a database of descriptors.

Discussion

Read more about fluid.datasetquery on the learn platform.

fluid.datasetquery is part of the Fluid Decomposition Toolkit of the FluCoMa project. For more explanations, learning material, and discussions on its musicianly uses, visit flucoma.org.

Arguments

Attributes

warnings [number] (default: 0)

Enable warnings to be issued whenever a parameter value is constrained (e.g. clipped)

Messages

transform

Arguments: sourceDataSet (Unknown Type (input dataset)), destDataSet (symbol)

Apply the query to a source fluid.dataset and write to a destination. Can be passed the same for input and output (in-place).

Argument details:
  • sourceDataSet:

    Source data, or the DataSet name

  • destDataSet:

    Destination data, or the DataSet name

transformjoin

Arguments: source1DataSet (Unknown Type (input dataset)), source2DataSet (Unknown Type (input dataset)), destDataSet (symbol)

Apply the query to a source FluidDataSet and join the resulting subset at the end of the items sharing the same identifiers in a second source. Items unique to a source dataset will be ignored. To add items at the end of a dataset instead, see the 'merge' method of FluidDataSet.

Argument details:
  • source1DataSet:

    Source data, or the DataSet name

  • source2DataSet:

    Source data, or the DataSet name

  • destDataSet:

    Destination data, or the DataSet name

addcolumn

Arguments: column (int)

Add a column to the query

Argument details:
  • column:

    Column index

addrange

Arguments: start (int), count (int)

Add a range of columns to the query

Argument details:
  • start:

    First index

  • count:

    Number of columns

filter

Arguments: column (int), condition (symbol), value (number)

Filter rows according to some condition.

Argument details:
  • column:

    Column index

  • condition:

    Condition string. Possible values: "==", "!=", "<", "<=", ">", ">="

  • value:

    Condition value

Argument details:

    Condition string. Possible values: "==", "!=", "<", "<=", ">", ">="

  • value:

    Condition value

or

Arguments: column (int), condition (symbol), value (number)

Add a condition to an existing filter with an "or" connector.

Argument details:
  • column:

    Column index

  • condition:

    Condition string. Possible values: "==", "!=", "<", "<=", ">", ">="

  • value:

    Condition value

Arguments:
