fluid.normalize

Normalize a FluidDataSet

Description

Normalize a fluid.dataset or single point of data according to learnt minimum and maximum values.

Discussion

Read more about fluid.normalize on the learn platform.

fluid.normalize is part of the Fluid Decomposition Toolkit of the FluCoMa project. For more explanations, learning material, and discussions on its musicianly uses, visit flucoma.org.

Arguments

Name [number]

Optional
(default: )

Name for this instance, so that it can be reused or referred to by other data objects. If none is given, a name will be generated

Attributes

max [number] (default: 1.0)

Maximum output value, default 1

min [number] (default: 0.0)

Minimum output value, default 0

warnings [number] (default: 0)

Enable warnings to be issued whenever a parameter value is constrained (e.g. clipped)
